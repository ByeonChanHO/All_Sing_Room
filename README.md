# All_Sing_Room

## 앱 이름 : 모두의 노래방
## 1.개요
친구들 또는 가족끼리 노래방을 가게 되었을 때 방이 없어 다른 노래방에 가게 되는 경우 발생하는 동선낭비와 시간낭비를 줄이기 위해 이 앱을 선정하였다.
## 2.필요한 부분들
1.	관리자(노래방 사장님), 사용자가 쓰는 어플이 각각 존재 해야한다.
사용자 어플에 필요한 부분들
2.	동전 노래방과 시간제 노래방을 각각 찾을 수 있도록 도와준다.
3.	검색 서비스로 노래방을 찾아준다.
4.	노래방에 대한 정보와 몇몇 사진을 올려 시설이 어떠 한지 정보를 제공한다.
5.	GPS를 기반으로 하여 노래방의 위치를 표시해준다.
6.	노래방의 자리가 있는 지를 알 수 있도록 정보제공
7.	노래방의 대기사람이 몇 명 있는 지를 상시 보여준다.
8.	별점과 댓글을 이용하여 랭킹을 매기게 한다(이는 관리자의 의욕을 고취시킨다.)
10.	노래방에 쓰이는 관리 시스템과 연동할 수 있도록 한다.
11.	컴퓨터가 없는 노래방을 고려하여 폰 용으로 할 수 있는 관리자시스템을 만든다.
12.	로그인 시스템을 이용하여 각 노래방 정보를 올릴 수 있는 권한을 준다.
13.	노래방의 대기 인원이 몇 명인지 표시해줄 권한을 준다
(노래방 관리자가 버튼을 눌러 수를 높일 뿐이다.)
13.	예약하려는 사람의 간단한 정보를 확인가능하도록한다.(거절도 된다)
##3.대표적 기술.
1.	로그인 시스템
2.	검색 시스템
3.	GPS 시스템
4.	서버 구축(관리자 어플 – 사용자 어플)
5.	데이터 베이스 구축
6.	앱 디자인
